## [Resume Analyzer](https://github.com/PraneethTVSS/FIQ/blob/main/resume_parser.py)

### Dependencies
```
pip install tika
```

### Linux Dependencies
```
apt install default-jre
apt install default-jdk
```

#### Run command:
```
python resume_parser.py <absolute path of the resume>
```
